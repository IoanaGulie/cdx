CDX Library

The example can be run with: python ./example-01.py cdx-convert-continuous-to-discrete -e -b 0.1e-6 -a 0.1e-6 -s 1000e6 -i example-01.cdx -o example-01.cdd cdx-display -i example-01.cdx

If you would like to run this script with a CDX library version in your home directory, set e.g.

PYTHONPATH=~/workspace/cdx/trunk/python python ./example-01.py

before calls to this script.
