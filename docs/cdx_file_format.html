The Channel Data Exchange (CDX) File Format

File Structure

All data is stored in groups within an HDF5 file. Groups resemble (sub-)directories present in file systems. The format is as follows:

Common Datasets

Group /parameters/:

HDF5 Entity Type Description
Group /parameters/ Group Contains parameters of the CDX file
/parameters/creation_time Float Creation time of the channel data
/parameters/delay_type String Either 'continuous-delay' or 'discrete-delay'
/parameters/c0_m_s Float The speed of light in m/s
/parameters/cir_rate_Hz Float The rate at which the channel impulse responses are saved in Hz
Group /parameters/channel/ (optional) Group For all channel model specific parameters, e.g. scenery definitions
/links/<link_name>/reference_ranges Matrix Reference ranges

Discrete-Delay CDX files

Group /parameters/:

HDF5 Entity Type Description
/parameters/number_of_delay_samples Integer This parameter defines the (constant) number of delay samples

Group /links/:

HDF5 Entity Type Description
/links/<link_name> Group Contains channel response for <link_name1>
/link0/<link_name>/cir_real Matrix Real part of the channel response
/link0/<link_name>/cir_imag Matrix Imaginary part of the channel response
